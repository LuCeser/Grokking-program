# 1. Algorithm: leetCode.234 回文链表

## 1.1 题目描述

判断一个链表是否为回文链表

比如:

`1->2` 为false
`1->2->2->1` 为true

这道题考察的知识点是链表。

## 1.2 解题思路

使用快慢指针

![快慢指针](images/快慢指针-奇数.jpg)



![快慢指针-偶数](images/快慢指针-偶数.jpg)


```python
# @lc code=start
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def isPalindrome(self, head: ListNode) -> bool:

        if not head:
            return True

        # 定义快指针慢指针
        fast = slow = head

        # 快指针移动两个，慢指针移动一格
        while fast:
            slow = slow.next
            fast = fast.next.next if fast.next else fast.next

        p, rev = slow, None
        # 反转中间列表之后的链表
        while p:
            rev, rev.next, p = p, rev, p.next

        # 从头开始将Head 与反转后的列表做对比
        while rev:
            if rev.val != head.val:
                return False
            rev = rev.next
            head = head.next
        return True
```
